### 项目简介
本项目是一个基于`SpringBoot` 和 `Vue3`的前后端分离项目，项目中数据库的配置均已经使用`TODO`进行标记

本项目不提供jar包。

项目部署地址[SmartCare](http://8.152.100.169)


项目接口文档[SmartCareApi](http://8.152.100.169:8080/doc.html)


整个项目部署在一台`2G 2核`服务器上， 同时存在`jar` `redis` `mysql` `minio` `ngnix` `web-view`
请大佬手下留情切勿进行高频访问，服务器属实扛不住 
### 环境配置
* jdk 17+
* mysql
* minio
### 坦白
本项目只是一个小小的crud项目，项目亮点可以说一点都没有， 仅仅可能只有在数据处理上面使用了`@Scheduled`和mysql的聚合函数。

这是一个有真实应用场景的项目，目前没有移动端和用户端的打算，所以`web-app`是一个空项目。
目前版本处于v1，后续可能会有迭代。
### 项目展望
* springcloud 分布式架构
* redis 缓存优化
* mysql集群
